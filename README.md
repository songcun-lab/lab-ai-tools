## 🎯 Why this lab exists

这个仓库不是用来「评测工具好不好」，  
而是用来记录：

- **我在真实任务中如何使用 AI 工具**
- **哪些工具真的改变了效率，哪些只是噱头**
- **非计算机背景的人，如何把 AI 用进工作流**

这里不追新，不追全，只追 **真实、可复用、可判断**。
